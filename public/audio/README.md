# Audio Files

Place the following MP3 files in this directory to enable in-game sound.

The game works perfectly without any of these files — audio is optional and
must be explicitly enabled by the user via the "🔊 Enable Sound" button.

---

## Background Music (loop-friendly, ~1–3 min each)

| File | When it plays |
|------|--------------|
| `bg-answering.mp3` | During answer submission phase |
| `bg-voting.mp3` | During voting phase |
| `bg-results.mp3` | During round results reveal |
| `bg-leaderboard.mp3` | During between-round leaderboard |
| `bg-gameover.mp3` | During final game-over screen |

Recommended: energetic, upbeat party/game-show music. Volume is set to 28%.

---

## Sound Effects (short clips, < 2 sec)

| File | When it plays |
|------|--------------|
| `sfx-answer-submit.mp3` | Player submits their answer |
| `sfx-vote.mp3` | Player taps to cast a vote |
| `sfx-reveal.mp3` | Each leaderboard position reveal (pop/whoosh) |
| `sfx-winner.mp3` | 1st place revealed / game winner fanfare |

---

## Where to get free audio

- **SFX**: [freesound.org](https://freesound.org), [pixabay.com/sound-effects](https://pixabay.com/sound-effects/)
- **Music**: [incompetech.com](https://incompetech.com), [freemusicarchive.org](https://freemusicarchive.org)

Recommended format: **MP3**, 44.1 kHz, stereo  
Target sizes: music ~2–5 MB, SFX ~50–200 KB each
