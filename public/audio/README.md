# Audio Assets

All audio files are committed to this directory and ready to use.
The "Enable Sound 🔊" button must be clicked by the user first (browser autoplay policy).
The game works perfectly even if audio is disabled or a file is missing.

## Format notes

- Background music: **MP3** — works on all browsers including iOS/Safari
- Sound effects: **OGG** — works on Chrome, Firefox, Android Chrome
  - iOS/Safari skips OGG silently; the game remains fully functional

---

## Replacing files

Drop in your own files with the same names to swap any track.
Recommended specs: MP3/OGG, 44.1 kHz, stereo, music ~2–5 MB, SFX ~50–200 KB.
